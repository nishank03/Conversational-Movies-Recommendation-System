# Conversational-Movies-Recommendation-System_1
About LLM-powered conversational movie recommender with three engines (Few-Shot, RAG with FAISS+BM25 hybrid retrieval, ReAct Agent), real-time voice interaction via ElevenLabs STT/TTS, streaming responses, and offline evaluation with Hit/Recall/MRR/NDCG metrics. Built with FastAPI, Anthropic Claude / OpenAI GPT, and sentence-transformers
